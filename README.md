# Iris
项目概述

本项目基于经典的Iris鸢尾花数据集，实现了多种机器学习分类器的训练、评估和可视化。通过4个任务，展示了从数据探索、分类器比较到高级三维可视化的完整机器学习工作流程。

环境要求与兼容性说明

⚠️ 重要：库版本兼容性问题

在运行本项目时，必须特别注意库版本的兼容性，特别是NumPy与其他科学计算库之间的版本匹配。我们遇到了典型的二进制不兼容问题：ValueError: numpy.dtype size changed, may indicate binary incompatibility.Expected 96 from C header, got 88 from PyObject

强烈推荐使用虚拟环境!!!!!!!!

任务1：可视化不同分类器的结果（三分类/两个特征）

文件： task1_multiclass.py

目标： 比较多种分类器在Iris数据集上的表现

任务2：可视化3D Boundary（两分类/三个特征）

文件： task2_3d_boundary.py

目标： 在三维空间中可视化决策边界

任务3：可视化3D Probability Map（两分类/三个特征）

文件： task3_3d_probability.py

目标： 可视化分类器在三维空间中的概率分布

任务4：高级可视化（加分项）

文件： task4_advanced_visualization.py

🎯 任务4创新点说明

1、多分类器性能比较

系统比较了SVM、随机森林和神经网络三种分类器

使用5折交叉验证评估性能

所有分类器在简化数据集上都达到了100%准确率

2、三维决策边界与概率图的融合

将3D决策边界（等值面）与概率云图结合展示

使用透明度表示概率密度

实现了多视角观察

3、特征对的二维投影分析

对三个特征的所有组合进行二维投影

每个投影显示分类概率的等高线图

便于理解特征之间的相互作用
