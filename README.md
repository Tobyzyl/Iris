# Iris
项目概述

本项目基于经典的Iris鸢尾花数据集，实现了多种机器学习分类器的训练、评估和可视化。通过4个任务，展示了从数据探索、分类器比较到高级三维可视化的完整机器学习工作流程。


环境要求与兼容性说明

⚠️ 重要：库版本兼容性问题

在运行本项目时，必须特别注意库版本的兼容性，特别是NumPy与其他科学计算库之间的版本匹配。我们遇到了典型的二进制不兼容问题：ValueError: numpy.dtype size changed, may indicate binary incompatibility.Expected 96 from C header, got 88 from PyObject

强烈推荐使用虚拟环境!!!!!!!!
